# nuk-csie-114-project

## 環境配置

### 作業系統

- Ubuntu 22.04 或 WSL2

### 安裝套件

- gcc/g++
```bash
sudo apt install build-essential
```

- python3
```bash
sudo apt install python3 python3-pip
```

- python3-venv (可選)
```bash
sudo apt install python3-venv
```

- cpplint
```bash
sudo apt install cpplint
```

- valgrind
```bash
sudo apt install valgrind
```

## 編譯與執行

```bash
python3 test.py
```
